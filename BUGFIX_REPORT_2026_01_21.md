## Исправленные проблемы

### 1. ✅ КРИТИЧЕСКАЯ: Рассинхронизация состояния подключения

**Проблема:**  
Сервер считал устройство подключенным, а клиент - нет (`_isConnected = false`). Это приводило к ошибке "Устройство уже подключено" при попытке подключения.

**Причина:**  
Метод `_syncConnectionState()` вызывался только при инициализации сервиса, но не перед каждым подключением.

**Решение:**  
Добавлен вызов `_syncConnectionState()` в метод `VpnService.connect()` перед этапом получения конфигурации (после проверки токена).

**Файлы:**  
- `mobile-app/lib/services/vpn_service.dart` (строки 743-748)

**Тесты:**  
- `test/services/vpn_service_test.dart` - группа "VpnService - синхронизация состояния подключения"

---

### 2. ✅ КРИТИЧЕСКАЯ: Недостаточная обработка ошибки "Устройство уже подключено"

**Проблема:**  
После принудительного отключения устройства на сервере повторная попытка подключения не срабатывала из-за отсутствия задержки для обновления состояния на сервере.

**Решение:**  
1. Добавлена задержка 500ms после отключения устройства
2. Добавлена повторная синхронизация состояния после отключения
3. Улучшена обработка ошибок - метод `_forceDisconnectOnServer()` теперь возвращает `bool` вместо `void`

**Файлы:**  
- `mobile-app/lib/services/vpn_service.dart` (строки 829-860, 1978-2020)

**Тесты:**  
- `test/services/vpn_service_test.dart` - группа "VpnService - обработка ошибки 'Устройство уже подключено' с задержкой"
- `test/services/vpn_service_test.dart` - группа "VpnService - принудительное отключение на сервере"

---

### 3. ✅ ВАЖНАЯ: Некорректная логика проверки истечения токена

**Проблема:**  
Токен считался истекшим за 1 день до реального истечения, что приводило к постоянным обновлениям токена и лишним запросам к серверу.

**Решение:**  
Изменен запас времени с 1 дня на 5 минут. Теперь токен обновляется только за 5 минут до истечения.

**Файлы:**  
- `mobile-app/lib/services/auth_service.dart` (строка 286)

**До:**  
```dart
final isExpired = now.isAfter(expiresAt.subtract(const Duration(days: 1)));
```

**После:**  
```dart
final isExpired = now.isAfter(expiresAt.subtract(const Duration(minutes: 5)));
```

**Тесты:**  
- `test/services/auth_service_test.dart` - группа "AuthService - проверка истечения токена"
- `test/services/auth_service_test.dart` - группа "AuthService - обновление токена"

### 4. ✅ ВАЖНАЯ: Отсутствие OnBackInvokedCallback в AndroidManifest

**Проблема:**  
В логах появлялось предупреждение о том, что `OnBackInvokedCallback` не включен для приложения.

**Решение:**  
Добавлен атрибут `android:enableOnBackInvokedCallback="true"` в тег `<application>` манифеста.

**Файлы:**  
- `mobile-app/android/app/src/main/AndroidManifest.xml` (строка 28)

**До:**  
```xml
<application
    android:label="@string/app_name"
    android:name="${applicationName}"
    android:icon="@mipmap/ic_launcher"
    android:usesCleartextTraffic="true">
```

**После:**  
```xml
<application
    android:label="@string/app_name"
    android:name="${applicationName}"
    android:icon="@mipmap/ic_launcher"
    android:usesCleartextTraffic="true"
    android:enableOnBackInvokedCallback="true">
```

### 5. ✅ СРЕДНЯЯ: Ошибка декодирования изображения флага

**Проблема:**  
В логах появлялась ошибка `ImageDecoder$DecodeException` при загрузке флагов стран.

**Решение:**

3. Добавлена оптимизация памяти через `memCacheWidth` и `memCacheHeight`

**Файлы:**  
- `mobile-app/lib/widgets/vpn/server_card.dart` (строки 49-60)

**Улучшения:**

- HTTP заголовки для поддержки разных форматов изображений
- Оптимизация использования памяти

### 6. ✅ СРЕДНЯЯ: Игнорирование ошибок при отключении на сервере

**Проблема:**  
Метод `_forceDisconnectOnServer()` игнорировал все ошибки, что могло скрывать реальные проблемы.

**Решение:**  
1. Метод теперь возвращает `bool` вместо `void`
2. Добавлена детальная обработка различных статус-кодов:
   - `200`, `204` - успешное отключение
   - `404`, `403` - устройство уже отключено (считается успехом)
   - Другие ошибки - логируются и возвращают `false`
3. Улучшено логирование для отладки

**Файлы:**  
- `mobile-app/lib/services/vpn_service.dart` (строки 1978-2020)

**Тесты:**  
- `test/services/vpn_service_test.dart` - группа "VpnService - принудительное отключение на сервере"
