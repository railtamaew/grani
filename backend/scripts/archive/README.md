# Архив скриптов

Эта папка содержит старые, временные или неиспользуемые скрипты.

## Принципы архивации

Скрипты перемещаются в архив, если:
1. Они были временными диагностическими инструментами
2. Они больше не используются в продакшене
3. Они были заменены более новыми версиями
4. Они использовались для разовых задач

## Восстановление

Если нужно восстановить скрипт из архива:
1. Проверьте, что он действительно нужен
2. Переместите обратно в основную папку `scripts/`
3. Обновите документацию в `scripts/README.md`

## Удаление

Перед удалением скриптов из архива:
- Убедитесь, что они точно не нужны
- Проверьте историю git для восстановления при необходимости
- Удаляйте только после согласования с командой
