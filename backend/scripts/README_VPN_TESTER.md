# VPN Tester - Интерактивный инструмент для тестирования VPN

## Описание

Интерактивный инструмент для тестирования VPN подключений прямо в Cursor. Позволяет:
- Просматривать список доступных серверов
- Подключаться и отключаться от VPN
- Тестировать разные протоколы
- Проверять статус подключения
- Измерять скорость и ping
- Просматривать статистику подключений
- Автоматически тестировать все протоколы на сервере

## Установка зависимостей

```bash
# Убедитесь, что установлены необходимые пакеты
pip install requests
```

## Использование

### Базовое использование

```bash
cd /opt/grani/backend
python3 scripts/vpn_tester.py
```

### С параметрами

```bash
# С другим пользователем
python3 scripts/vpn_tester.py --email your@email.com

# С другим API URL (для продакшена)
python3 scripts/vpn_tester.py --api-url https://api.granilink.com

# Комбинация
python3 scripts/vpn_tester.py --email rail.tamaew@gmail.com --api-url http://localhost:8000
```

## Примеры использования

### 1. Просмотр списка серверов

Выберите `1` в меню. Вы увидите список всех доступных серверов с их протоколами.

### 2. Подключение к VPN

1. Выберите `2` в меню
2. Выберите ID сервера из списка
3. Выберите протокол (или нажмите Enter для протокола по умолчанию)

### 3. Проверка статуса

Выберите `4` в меню для просмотра текущего статуса подключения.

### 4. Тест всех протоколов

Выберите `7` в меню и введите ID сервера. Инструмент автоматически протестирует все поддерживаемые протоколы на этом сервере.

### 5. Тест скорости

Выберите `5` в меню. Инструмент выполнит ping тест. Для полного теста скорости рекомендуется использовать `speedtest-cli`:

```bash
pip install speedtest-cli
speedtest-cli
```

## Поддерживаемые протоколы

- `wireguard` - WireGuard
- `graniwg` - GRANIWG
- `xray_vless` - Xray VLESS
- `xray_vmess` - Xray VMESS
- `xray_reality` - Xray Reality
- `openvpn_cloak` - OpenVPN over Cloak

## Требования

- Python 3.7+
- Доступ к базе данных
- Запущенный backend API
- Пользователь должен быть зарегистрирован в системе

## Устранение неполадок

### Ошибка "Пользователь не найден"

Убедитесь, что пользователь зарегистрирован в системе:
```bash
python3 scripts/test_token_servers.py
```

### Ошибка подключения к API

Проверьте, что backend запущен:
```bash
# Проверка статуса
curl http://localhost:8000/api/health

# Или используйте другой URL
python3 scripts/vpn_tester.py --api-url https://api.granilink.com
```

### Ошибка регистрации устройства

Устройство автоматически регистрируется при первом использовании. Если возникает ошибка, проверьте логи backend.

## Безопасность

- Токен автоматически генерируется для указанного пользователя
- Device ID генерируется автоматически для каждого сеанса
- При выходе активное подключение автоматически отключается

## Дополнительные возможности

### Автоматическое тестирование

Для автоматического тестирования всех протоколов на всех серверах можно создать скрипт-обертку:

```python
from scripts.vpn_tester import VPNTester

tester = VPNTester()
tester.get_token()
tester.register_device()

servers = tester.get_servers()
for server in servers:
    if server.get('is_active'):
        tester.test_all_protocols(server['id'])
```
