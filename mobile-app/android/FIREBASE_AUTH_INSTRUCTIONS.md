# 🔐 Инструкция по авторизации для Firebase Test Lab

## Текущее состояние

- ✅ Проект настроен: `grani-479612`
- ✅ API включены
- ✅ APK готовы
- ⚠️ Требуется авторизация пользовательским аккаунтом

## Шаг 1: Авторизация пользовательским аккаунтом

Выполните следующую команду в терминале на сервере:

```bash
gcloud auth login --no-browser
```

Команда выведет ссылку вида:
```
Go to the following link in your browser:
https://accounts.google.com/o/oauth2/auth?...
```

### Что делать дальше:

1. **Скопируйте ссылку** из вывода команды
2. **Откройте ссылку** в браузере на вашем локальном компьютере
3. **Авторизуйтесь** через ваш Google аккаунт (который имеет доступ к проекту `grani-479612`)
4. **Скопируйте код авторизации** из браузера
5. **Вставьте код** в терминал на сервере

### Альтернативный вариант (если есть графический интерфейс):

```bash
gcloud auth login
```

Откроет браузер автоматически.

## Шаг 2: Проверка авторизации

После успешной авторизации проверьте:

```bash
gcloud auth list
```

Должен показать ваш пользовательский аккаунт (email) как активный.

## Шаг 3: Проверка настройки

```bash
cd /opt/grani/mobile-app/android
./scripts/check_firebase_setup.sh
```

Должно показать:
- ✅ Аккаунт: ваш_email@gmail.com (вместо сервисного аккаунта)
- ✅ Проект: grani-479612
- ✅ API включены
- ✅ APK готовы

## Шаг 4: Запуск тестов

После успешной авторизации и проверки:

```bash
cd /opt/grani/mobile-app/android

./scripts/run_firebase_tests.sh \
    --singbox-config test_configs/singbox_minimal.json \
    --device-model Pixel2.arm \
    --device-version 30
```

## Просмотр результатов

После завершения тестов результаты будут доступны:
https://console.firebase.google.com/project/grani-479612/testlab/histories

## Важные замечания

1. **Права доступа:** Убедитесь, что ваш Google аккаунт имеет доступ к проекту `grani-479612`
2. **Если доступа нет:** Попросите администратора проекта добавить вас:
   ```bash
   gcloud projects add-iam-policy-binding grani-479612 \
       --member="user:ваш_email@gmail.com" \
       --role="roles/firebase.admin"
   ```
