# AmneziaWG binaries

Place amneziawg-go binaries here for desktop GraniWG with obfuscation:

- `macos/arm64/amneziawg-go` — macOS Apple Silicon
- `macos/amd64/amneziawg-go` — macOS Intel
- `windows/awg-quick.exe` — Windows GRANIwg/AmneziaWG runner used by the
  Flutter Windows MethodChannel

For Windows development without bundling, set `GRANI_AWG_QUICK` to the full
path of a compatible `awg-quick.exe`.

Build from: https://github.com/amnezia-vpn/amneziawg-go
