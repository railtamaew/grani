# VPN Minimal Test Application

Минимальное приложение для тестирования VPN протоколов снизу вверх.

## 🎯 Цель

Тестировать VPN протоколы изолированно, не затрагивая основное приложение.

## 🚀 Быстрый старт

### 1. Сборка APK

```bash
cd /opt/grani/test-projects/vpn-minimal
flutter build apk --debug
```

### 2. Установка на устройство

```bash
adb install build/app/outputs/flutter-apk/app-debug.apk
```

### 3. Запуск логирования

```bash
adb logcat -c  # Очистка логов
adb logcat | grep -E "MinimalVpn|Flutter" > /tmp/vpn_test_$(date +%Y%m%d_%H%M%S).log &
```

### 4. Тестирование

1. Откройте приложение на устройстве
2. Нажмите "Подключиться"
3. Разрешите VPN подключение (если запросит)
4. Наблюдайте логи
5. Нажмите "Отключиться"

## 📝 Логи

Логи сохраняются в `/tmp/vpn_test_*.log` и содержат:
- Все действия VPN сервиса
- Ошибки подключения
- Статусы подключения
