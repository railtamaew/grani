## 🧪 Шаг 2: Тестирование базовой функциональности

### 1. Запустите приложение на устройстве

- Найдите "VPN Minimal Test" в списке приложений
- Откройте приложение

### 2. Проверьте логирование в реальном времени

На сервере откройте терминал и выполните:

```bash
# Просмотр логов в реальном времени
tail -f /opt/grani/logs/test_apps/vpn-minimal_$(date +%Y%m%d).log
```

### 3. Тестирование подключения

1. **Нажмите "Подключиться"** в приложении
2. **Разрешите VPN подключение** (если система запросит)
3. **Проверьте статус:**
   - В приложении должно показать "Подключено"
   - В уведомлениях должно появиться "VPN подключен"
4. **Проверьте логи на сервере:**
   - Должны появиться записи о подключении
   - Формат: `timestamp | INFO | VPN | Начало подключения VPN`
   - Формат: `timestamp | INFO | VPN | VPN подключен успешно`

### 4. Тестирование отключения

1. **Нажмите "Отключиться"** в приложении
2. **Проверьте статус:**
   - В приложении должно показать "Отключено"
   - Уведомление должно исчезнуть
3. **Проверьте логи на сервере:**
   - Должны появиться записи об отключении
   - Формат: `timestamp | INFO | VPN | Начало отключения VPN`
   - Формат: `timestamp | INFO | VPN | VPN отключен успешно`

## ❌ Что проверить при ошибках

### Логи не приходят на сервер

1. Проверьте интернет-соединение на устройстве
2. Проверьте, что API доступен: `curl http://159.223.199.122:8000/api/vpn/logs/test`
3. Проверьте логи приложения через ADB:
   ```bash
   adb logcat | grep SimpleLogger
   ```

### VPN не подключается

1. Проверьте разрешения VPN в настройках Android
2. Проверьте логи через ADB:
   ```bash
   adb logcat | grep MinimalVpnService
   ```
3. Убедитесь, что нет других активных VPN подключений

### Приложение не запускается

1. Проверьте логи через ADB:
   ```bash
   adb logcat | grep -E "AndroidRuntime|FATAL"
   ```
2. Переустановите приложение:
   ```bash
   adb uninstall com.granivpn.test.vpn_minimal
   adb install build/app/outputs/flutter-apk/app-debug.apk
   ```
