# Деплой админ-панели на admin.granilink.com

Чтобы на продакшене отображались последние изменения (тайтл «GRANI Admin», фавикон, логотип, исправление удаления устройств), нужно задеплоить и фронт, и бэкенд.

## 1. Фронт (админ-панель)

1. Собрать проект:
   ```bash
   cd /opt/grani/admin-panel
   npm run build
   ```
2. Выложить содержимое папки **`build/`** на хостинг, который обслуживает **admin.granilink.com**:
   - в корень сайта должны попасть: `index.html`, `favicon.png`, `logo.png`, `runtime-config.js`, папка `static/`
   - при использовании nginx/apache убедитесь, что для SPA настроен fallback на `index.html` для всех маршрутов

## 2. Бэкенд (удаление устройств)

Код с исправлением удаления устройств находится в `backend/api/admin.py`. Нужно задеплоить обновлённый бэкенд на сервер, к которому обращается admin.granilink.com (тот же сервер или отдельный API).

После деплоя:
- тайтл вкладки: **GRANI Admin**
- фавикон: щит с буквой G
- логотип в сайдбаре: только картинка (без зелёного квадрата и отдельной буквы G)
- удаление устройств (одиночное и массовое) должно проходить без ошибки
