# Краткая сводка: AmneziaVPN vs GRANI VPN

## 🎯 Ключевые отличия

| Аспект | AmneziaVPN | GRANI VPN | Победитель |
|--------|-----------|-----------|------------|
| **Архитектура** | Docker контейнеры | Прямая установка | GRANI (меньше overhead) |
| **Обфускация WireGuard** | AmneziaWG (продвинутая) | GRANIWG (заглушка) | AmneziaVPN |
| **Протоколы** | 6 протоколов | 6 протоколов | Ничья |
| **Управление** | Через клиент | Через backend API | GRANI (централизованное) |
| **Мониторинг** | Базовый | Продвинутый | GRANI |
| **Производительность** | Docker overhead | Нативная | GRANI |
| **Масштабируемость** | Docker изоляция | Централизованное | Ничья |

---

## ✅ Что делает GRANI VPN лучше

1. **Централизованное управление** - backend API управляет всеми серверами
2. **Мониторинг и статистика** - встроены в систему
3. **Производительность** - нет Docker overhead
4. **XRay протоколы** - полностью реализованы (VLESS, VMESS, REALITY)
5. **Гибкость** - поддержка локальных и удаленных серверов

---

## ❌ Что нужно улучшить в GRANI VPN

1. **GRANIWG обфускация** - критично! Сейчас только заглушка
2. **Обфускация WireGuard** - менее продвинутая чем AmneziaWG
3. **Протоколы** - добавить IKEv2, улучшить Shadowsocks поддержку

---
