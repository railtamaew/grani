# Тестирование Xray протоколов на реальном устройстве

## Подготовка

### 1. Получение конфигураций

Используйте скрипт для экспорта конфигураций:

```bash
cd /opt/grani/backend
python3 scripts/export_client_configs.py --code YOUR_CODE
```

Скрипт создаст файлы с конфигурациями в директории `configs/`.

### 2. Установка VPN клиента на устройство

Рекомендуемые клиенты для Android:
- **v2rayNG** - лучший выбор для Xray протоколов
- **V2ray** - альтернативный клиент
- **SingBox** - универсальный клиент

Для iOS:
- **Shadowrocket**
- **Quantumult X**

## Тестирование протоколов

### VLESS

1. Скопируйте URL конфигурации из файла `vless_*.conf`
2. Откройте v2rayNG
3. Нажмите "+" → "Импорт из буфера обмена"
4. Вставьте URL конфигурации
5. Нажмите "Подключиться"

### VMESS

1. Скопируйте URL конфигурации из файла `vmess_*.conf`
2. Откройте v2rayNG
3. Нажмите "+" → "Импорт из буфера обмена"
4. Вставьте URL конфигурации
5. Нажмите "Подключиться"

### REALITY

1. Скопируйте URL конфигурации из файла `reality_*.conf`
2. Откройте v2rayNG
3. Нажмите "+" → "Импорт из буфера обмена"
4. Вставьте URL конфигурации
5. Нажмите "Подключиться"

**Примечание:** REALITY может требовать дополнительных настроек в клиенте.

## Проверка работы

После подключения проверьте:

1. **Статус подключения** - должна быть активная VPN сессия
2. **IP адрес** - должен измениться на IP сервера (45.12.132.94)
3. **Скорость интернета** - проверьте через speedtest
4. **Доступность сайтов** - проверьте доступ к заблокированным ресурсам

## Использование QR-кода

Для удобства можно использовать QR-код:

1. Откройте URL конфигурации в браузере
2. Сгенерируйте QR-код (используйте онлайн генератор)
3. Отсканируйте QR-код в v2rayNG

## Получение конфигурации через API

Также можно получить конфигурацию напрямую через API:

```bash
# 1. Получить токен
TOKEN=$(curl -s -X POST "http://localhost:8000/api/auth/verify-code" \
  -H "Content-Type: application/json" \
  -d '{"email": "rail.tamaew@gmail.com", "code": "YOUR_CODE"}' \
  | jq -r '.access_token')

# 2. Создать клиента и получить конфигурацию
curl -X POST "http://localhost:8000/api/vpn/xray/create-client" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  -d '{
    "server_id": 1,
    "device_id": "your-device-id",
    "protocol": "xray_vless"
  }' | jq -r '.config'
```

## Решение проблем

### Конфигурация не импортируется

- Проверьте формат URL (должен начинаться с `vless://` или `vmess://`)
- Убедитесь, что URL не обрезан
- Попробуйте использовать JSON конфигурацию

### Подключение не работает

- Проверьте статус сервера: `systemctl status xray`
- Проверьте, что порт 443 открыт
- Проверьте логи Xray на сервере

### Медленная скорость

- Попробуйте другой протокол (REALITY обычно быстрее)
- Проверьте загрузку сервера
- Проверьте качество интернет-соединения
