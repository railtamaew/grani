## 2. Возможные причины «не работает ни первое, ни реконнект»

### A) Выбран протокол VMESS вместо VLESS
Если в приложении выбран **Xray VMESS**, а на сервере этот UUID не добавлен в VMESS inbound, получим `user do not exist`.

**Проверка:**  
Выберите **Xray VLESS** и попробуйте подключиться снова.

### B) Кэш конфига и смена протокола
Кэш: `xray_config_${serverId}_${protocol}`.  
При смене VLESS → VMESS (или обратно) используется другой ключ, новый create-client. Если create-client для VMESS не выполнился или VMESS inbound не настроен, будет отказ.

**Рекомендация:**  
При проблемах — полное отключение VPN, очистка данных приложения (или выход из аккаунта) и повторное подключение с нужным протоколом.

### C) Слишком частые рестарты XRay
Каждый create-client вызывает reload XRay. В момент reload уже установленные соединения могут рваться.

Если create-client вызывается при каждом connect (например, из‑за сброса кэша), это может приводить к нестабильности.

### D) Ошибка до сервера
Если в логах нет `accepted` для вашего client_id после попытки подключения, трафик может не доходить до XRay (сеть, фаервол, DNS, неправильный адрес/порт).

## 3. Что проверить

### 3.1 В приложении
1. **Протокол:** убедитесь, что выбран **Xray VLESS**, а не VMESS или REALITY.
2. **Очистка кэша:** при необходимости удалите данные приложения или выйдите из аккаунта и войдите снова.
3. **Сеть:** проверьте работу без VPN (Wi‑Fi, мобильный интернет).

### 3.2 Логи приложения (adb logcat)
```bash
adb logcat | grep -E "VpnService|XrayProtocol|GraniVpnService|XrayNativeWrapper"
```

Обратите внимание на:
- `create-client` vs `Используем конфиг из кэша`
- Ошибки вида `Проверка подключения не пройдена`, `Не удалось установить подключение`
- `DioException`, таймауты, коды ответа API

### 3.3 Повторная проверка логов сервера
```bash
docker exec granivpn_api python3 /app/scripts/fetch_server_logs.py --name HU-BUD-01 --xray-lines 100
```

Проверьте, появляются ли в `access.log` записи `accepted` с вашим client_id (email) в момент попытки подключения.
