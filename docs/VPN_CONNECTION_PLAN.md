# Детальный план по подключению VPN

**Дата создания:** 2025-01-XX  
**Статус:** В разработке  
**Версия:** 1.0

## Обзор

План описывает полный процесс реализации VPN подключения в приложении GRANI, включая backend подготовку, интеграцию с мобильным приложением и тестирование.

## Архитектура

### Компоненты системы

1. **Backend (FastAPI)**
   - API эндпоинты для управления VPN
   - ConnectionManager - управление подключениями
   - RemoteVPNManager - управление удаленными серверами через SSH
   - WireGuardManager - управление WireGuard
   - SSHManager - SSH подключения к серверам

2. **Mobile App (Flutter)**
   - VpnService - сервис управления VPN
   - NativeVpnService - нативный VPN сервис (Android)
   - VpnPlugin - плагин для нативного VPN

3. **VPN Серверы**
   - Удаленные серверы с WireGuard/Xray
   - Управление через SSH ключи
   - Автоматическая настройка конфигураций

## API Эндпоинты

### POST /api/vpn/device/register

**Описание:** Регистрация устройства

**Запрос:**
```json
{
  "device_id": "unique_device_id",
  "name": "My Device",
  "platform": "android"
}
```

**Ответ:**
```json
{
  "message": "Устройство зарегистрировано",
  "device_id": 1
}
```

### POST /api/vpn/connect

**Описание:** Подключение к VPN

**Запрос:**
```json
{
  "server_id": 1,
  "device_id": "unique_device_id",
  "protocol": "wireguard"  // опционально
}
```

**Ответ:**
```json
{
  "success": true,
  "config": "[Interface]\nPrivateKey = ...",
  "ip_address": "10.0.0.2"
}
```

### POST /api/vpn/disconnect

**Описание:** Отключение от VPN

**Запрос:**
```json
{
  "device_id": "unique_device_id"
}
```

**Ответ:**
```json
{
  "success": true,
  "message": "Отключено"
}
```

### GET /api/vpn/servers

**Описание:** Список доступных серверов

**Ответ:**
```json
[
  {
    "id": 1,
    "name": "Server 1",
    "country": "US",
    "city": "New York",
    "ip_address": "45.12.132.94",
    "port": 51820,
    "current_users": 10,
    "max_users": 1000,
    "is_active": true,
    "health_status": "healthy",
    "supported_protocols": ["wireguard", "xray_vless"]
  }
]
```

### GET /api/vpn/status

**Описание:** Статус VPN подключения

**Параметры:**
- `device_id` - ID устройства

**Ответ:**
```json
{
  "connected": true,
  "ip_address": "10.0.0.2",
  "server": {
    "id": 1,
    "name": "Server 1"
  },
  "connected_since": "2025-01-01T00:00:00"
}
```
