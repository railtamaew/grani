## 📋 Конкретные шаги СЕЙЧАС

### Шаг 1: Протестировать XRay в основном приложении

```bash
# 1. Собрать APK основного приложения
cd /opt/grani/mobile-app/android
./gradlew assembleDebug

# 2. Установить на устройство
adb install -r app/build/outputs/apk/debug/app-debug.apk

# 3. Протестировать подключение через XRay VLESS
```

### Шаг 2: Исправить ошибку "Не удалось запустить VPN сервис"

**Проблема:** Сервис не инициализируется правильно

**Решение:**
- Добавить `onStartCommand` в `GraniVpnService.kt`
- Улучшить логику запуска сервиса
- Добавить больше логирования

### Шаг 3: Использовать unit-тесты для быстрой проверки

```bash

./gradlew test

# Если тесты прошли → собираем APK
./gradlew assembleDebug
```
