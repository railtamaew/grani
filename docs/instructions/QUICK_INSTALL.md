# Быстрая установка APK

## Если вы уже подключены через Cursor SSH

Если вы работаете через Cursor и уже подключены к серверу (видите файлы в `/opt/grani`), то скрипт можно запустить прямо из терминала Cursor:

```bash
/opt/grani/INSTALL_APK.sh
```

## Если вы в Windows PowerShell

### Вариант 1: Через SSH хост "grani" (если настроен)

```powershell
ssh grani "/opt/grani/INSTALL_APK.sh"
```

### Вариант 2: Через прямой IP адрес

```powershell
# Замените на ваш IP адрес сервера
ssh root@ваш-ip-адрес "/opt/grani/INSTALL_APK.sh"
```

### Вариант 3: Интерактивное подключение

```powershell
# 1. Подключитесь
ssh root@ваш-ip-адрес
# или
ssh grani

# 2. После подключения выполните:
/opt/grani/INSTALL_APK.sh
```

## Что делает скрипт

1. ✅ Проверяет подключенные Android устройства
2. ✅ Находит или собирает APK файл
3. ✅ Устанавливает APK на ваше устройство (OnePlus)
4. ✅ Настраивает логирование
5. ✅ Запускает приложение

## Требования

- Android устройство подключено по USB с включенной отладкой
- Или запущен Android эмулятор
- ADB доступен на сервере

## Проверка результата

После выполнения вы увидите:
- ✅ Сообщение об успешной установке
- Адрес файла с логами
- Информацию об устройстве

## Просмотр логов

```bash
# В реальном времени
tail -f /opt/grani/logs/app_*.log

# Последние 50 строк
tail -50 /opt/grani/logs/app_*.log
```
