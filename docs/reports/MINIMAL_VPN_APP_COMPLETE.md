## 🚀 Как использовать

### Шаг 1: Сборка и установка

```bash
cd /opt/grani/test-projects/vpn-minimal
./scripts/build_and_install.sh
```

**Что делает скрипт:**
1. Проверяет подключение устройства
2. Собирает APK через Flutter
3. Устанавливает на устройство

### Шаг 2: Запуск с логированием

```bash
./scripts/run_with_logs.sh
```

**Что делает скрипт:**
1. Очищает старые логи
2. Запускает логирование в фоне
3. Открывает приложение на устройстве
4. Сохраняет логи в `/tmp/vpn_minimal_test_*.log`

### Шаг 3: Тестирование

1. **Откройте приложение** на устройстве
2. **Нажмите "Подключиться"**
   - Если запросит VPN разрешение - разрешите
   - Наблюдайте логи в реальном времени
3. **Проверьте статус:**
   - Должно показать "Подключено"
   - В логах должно быть: "✅ VPN интерфейс создан успешно"
4. **Нажмите "Отключиться"**
   - Должно показать "Отключено"
   - В логах должно быть: "✅ VPN отключен успешно"
