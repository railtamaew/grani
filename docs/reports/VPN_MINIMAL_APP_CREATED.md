## 📊 Процесс тестирования

### 1. Сборка и установка

```bash
cd /opt/grani/test-projects/vpn-minimal
./scripts/build_and_install.sh
```

### 2. Запуск с логированием

```bash
./scripts/run_with_logs.sh
```

### 3. Тестирование на устройстве

1. Откройте приложение
2. Нажмите "Подключиться"
3. Разрешите VPN (если запросит)
4. Наблюдайте логи
5. Нажмите "Отключиться"

### 4. Анализ результатов

- Проверить логи
- Проверить статус VPN
- Документировать результаты

## ✅ Преимущества подхода

1. **Основное приложение не затрагивается**
   - Все тесты в отдельном проекте
   - Ничего не сломается в основном приложении

2. **Изолированное тестирование**
   - Проблемы видны сразу
   - Легко отлаживать

3. **Постепенная интеграция**
   - Только проверенные изменения
   - Минимальный риск

4. **Полное покрытие тестами**
   - Каждый уровень протестирован
   - Документированы все результаты

## 📝 Важные замечания

1. **Физическое устройство обязательно**
   - VPN интерфейсы не работают в эмуляторе
   - Firebase Test Lab не поддерживает TUN

2. **Реальные конфигурации**
   - Использовать реальные ключи и серверы
   - Тестировать на реальном VPN сервере

3. **Логирование**
   - Все действия логируются
   - Логи сохраняются для анализа

4. **Документация**
   - Все результаты документируются
   - Известные проблемы фиксируются
