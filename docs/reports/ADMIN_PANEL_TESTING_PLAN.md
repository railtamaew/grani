## 1. Дашборд (`/dashboard`)

| № | Действие | Ожидаемый результат |
|---|----------|---------------------|
| 1.1 | Открыть страницу | Отображаются карточки: Всего пользователей, Активные подписки, Серверы, Платежи |
| 1.2 | Проверить график подключений | LineChart по дням недели (Пн–Вс) |
| 1.3 | Проверить pie chart провайдеров | Распределение по auth providers |
| 1.4 | Проверить алерты нагрузки | Список серверов с load > 80% (если есть) |
| 1.5 | Проверить мониторинг | Метрики и диагностика ping (или сообщение об отключении) |
| 1.6 | Нажать «Обновить» | Данные перезагружаются без ошибок |
