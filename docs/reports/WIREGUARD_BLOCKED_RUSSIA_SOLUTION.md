# Решение проблемы блокировки WireGuard в России

## Проблема

WireGuard **блокируется в России** через DPI (Deep Packet Inspection). Это подтверждается тем, что:
- ✅ Туннель подключается (handshake проходит)
- ❌ Сайты не открываются
- ❌ Трафик не проходит (получено очень мало данных)
- ❌ Даже на других VPN (Amnesia VPN) WireGuard не работает в России

## Решение: Использовать обфусцированные протоколы

В проекте уже реализована поддержка обфусцированных протоколов, которые обходят блокировки DPI:

### 1. XRay VLESS/VMESS (РЕКОМЕНДУЕТСЯ)
- ✅ **Уже готовы к использованию** на сервере
- ✅ Используют порт 443 (HTTPS) - не блокируются
- ✅ Обходят DPI через маскировку под HTTPS трафик
- ✅ Высокая скорость и стабильность

### 2. XRay Reality
- ⚠️ Требует настройки (нужен reality_public_key)
- ✅ Лучше всего обходит блокировки
- ✅ Маскируется под реальный сайт (например, google.com)

### 3. GRANIWG (обфусцированный WireGuard)
- ⚠️ Требует настройки (нужен ключ обфускации)
- ✅ Использует обфускацию UDP трафика

### 4. OpenVPN Cloak
- ⚠️ Требует настройки
- ✅ Обходит блокировки через маскировку

## Быстрое решение: Настроить XRay

### Шаг 1: Настройка XRay на сервере

```bash
cd /opt/grani/backend
python3 scripts/setup_xray_server.py
```

Этот скрипт:
- Установит XRay на сервер
- Создаст базовую конфигурацию
- Запустит XRay на порту 443

### Шаг 2: Обновление сервера в БД

Нужно обновить сервер, чтобы указать порт XRay:

```python
# Через API или скрипт
server.xray_port = 443
```

### Шаг 3: Использование в мобильном приложении

В мобильном приложении выберите протокол:
- `xray_vless` - рекомендуется
- `xray_vmess` - альтернатива

## Текущий статус протоколов на сервере

Проверка готовности:
```bash
cd /opt/grani/backend
python3 scripts/check_protocols_readiness.py
```

## Рекомендация

**Используйте XRay VLESS или VMESS** - они уже готовы и работают в России.

### Преимущества XRay:
1. ✅ Обходит блокировки DPI
2. ✅ Использует порт 443 (HTTPS) - не блокируется
3. ✅ Высокая скорость
4. ✅ Стабильное соединение
5. ✅ Уже настроен на сервере

### Недостатки WireGuard в России:
1. ❌ Блокируется DPI
2. ❌ Трафик не проходит даже при подключении
3. ❌ Не работает на большинстве VPN в России

## Следующие шаги

1. **Настроить XRay на сервере:**
   ```bash
   python3 scripts/setup_xray_server.py
   ```

2. **Обновить мобильное приложение:**
   - Выбрать протокол `xray_vless` или `xray_vmess`
   - Подключиться к серверу

3. **Протестировать:**
   - Открыть сайты в браузере
   - Проверить скорость
   - Убедиться, что интернет работает

## Альтернатива: Настроить XRay Reality

Если XRay VLESS/VMESS тоже будут блокироваться (маловероятно), можно настроить XRay Reality:

```bash
# Скрипт для настройки Reality (будет создан)
python3 scripts/setup_xray_reality.py
```

Reality маскируется под реальный сайт и лучше всего обходит блокировки.
