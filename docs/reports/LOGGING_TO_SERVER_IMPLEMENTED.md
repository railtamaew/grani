### 2. SimpleLogger в минимальном приложении

**Файл:** `test-projects/vpn-minimal/lib/services/simple_logger.dart`

**Функциональность:**
- ✅ Логирование с уровнями (DEBUG, INFO, WARN, ERROR)
- ✅ Накопление логов в памяти
- ✅ Автоматическая отправка каждые 30 секунд
- ✅ Немедленная отправка при накоплении 10+ логов
- ✅ Повторная попытка при ошибках

**Методы:**
- `log(level, tag, message, extra)` - общий метод
- `debug(tag, message, extra)` - для отладки
- `info(tag, message, extra)` - для информации
- `warn(tag, message, extra)` - для предупреждений
- `error(tag, message, extra)` - для ошибок
- `flush()` - принудительная отправка
- `setEnabled(enabled)` - включить/выключить

## 🚀 Как использовать

### 1. Запустить приложение

```bash
cd /opt/grani/test-projects/vpn-minimal
./scripts/build_and_install.sh
```

### 2. Логи автоматически отправляются

- При запуске приложения
- При подключении VPN
- При отключении VPN
- При ошибках

### 3. Просмотр логов на сервере

```bash
# В реальном времени
tail -f /opt/grani/logs/test_apps/vpn-minimal_$(date +%Y%m%d).log

# Поиск ошибок
grep "ERROR" /opt/grani/logs/test_apps/vpn-minimal_*.log

# Поиск по тегу
grep "VPN" /opt/grani/logs/test_apps/vpn-minimal_*.log
```

## ✅ Преимущества

1. **Не нужен ADB** - логи приходят автоматически
2. **Работает на любом устройстве** - не нужен USB
3. **История логов** - все сохраняется на сервере
4. **Удобный анализ** - можно фильтровать и искать
5. **Автоматическая отправка** - не нужно ничего делать вручную

## 🔧 Настройка

### Включить/выключить логирование:

```dart
final logger = SimpleLogger();
logger.setEnabled(false); // Выключить
logger.setEnabled(true);  // Включить
```

### Принудительная отправка:

```dart
final logger = SimpleLogger();
await logger.flush(); // Отправить все накопленные логи немедленно
```
