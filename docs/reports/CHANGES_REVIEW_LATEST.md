## 🔍 Технические детали

### Исправление фильтрации серверов

**Логика фильтрации:**
1. Проверяются поддерживаемые протоколы сервера
2. Если сервер поддерживает только WireGuard/GRANIWG:
   - Требуется наличие `wireguard_public_key`
3. Если сервер поддерживает другие протоколы (XRay, OpenVPN Cloak):
   - `wireguard_public_key` не обязателен
   - Проверяется наличие хотя бы одного валидного протокола

### Автоматическая загрузка серверов

**Процесс:**
1. Пользователь открывает bottom sheet выбора сервера
2. Проверяется наличие загруженных серверов
3. Если серверы не загружены:
   - Показывается индикатор загрузки
   - Выполняется запрос к API `/vpn/servers`
   - Серверы фильтруются по поддерживаемым протоколам
   - Показывается список серверов
