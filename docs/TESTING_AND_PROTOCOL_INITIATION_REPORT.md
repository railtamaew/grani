## 📝 Примечания

1. **SSH предупреждения:** При тестировании WireGuard появляются предупреждения о SSH подключении, но это не критично - конфигурация создается успешно.

2. **API аутентификация:** Скрипт `initiate_protocol_connections.py` требует валидный токен. Для полного тестирования через API необходимо использовать реальный сервер с правильной конфигурацией токенов.

3. **Прямое тестирование:** Скрипт `test_protocols_direct.py` обходит проблемы с API, тестируя протоколы напрямую через backend менеджеры.
