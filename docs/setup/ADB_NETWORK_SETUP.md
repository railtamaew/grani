# Настройка ADB over Network

## Быстрая настройка для OnePlus устройства

### Шаг 1: Подготовка устройства

1. **Включите режим разработчика:**
   - Настройки → О телефоне → Номер сборки
   - Нажмите 7 раз на "Номер сборки"
   - Появится сообщение "Вы стали разработчиком!"

2. **Включите отладку по USB:**
   - Настройки → Система → Для разработчиков
   - Включите "Отладка по USB" (USB debugging)

3. **Подключите устройство к компьютеру:**
   - Используйте USB кабель
   - На устройстве появится запрос "Разрешить отладку по USB?"
   - Поставьте галочку "Всегда разрешать с этого компьютера"
   - Нажмите "Разрешить"

### Шаг 2: Проверка подключения

Выполните на сервере:
```bash
adb devices
```

Должно показать:
```
List of devices attached
XXXXXXXX        device
```

### Шаг 3: Узнайте IP адрес устройства

**Способ 1: Через настройки Android**
- Настройки → Wi-Fi → Нажмите на текущую сеть
- Найдите "IP адрес" (например, `192.168.1.100`)

**Способ 2: Через ADB (после подключения)**
```bash
adb shell "ip addr show wlan0 | grep 'inet ' | awk '{print \$2}' | cut -d/ -f1"
```

**Способ 3: Через настройки**
- Настройки → О телефоне → Статус → IP адрес

### Шаг 4: Автоматическая настройка

Запустите скрипт:
```bash
/opt/grani/setup_adb_network.sh
```

Скрипт автоматически:
1. Найдет USB устройство
2. Получит IP адрес
3. Включит TCP/IP режим
4. Подключится по сети

### Шаг 5: Ручная настройка (если автоматическая не работает)

```bash
# 1. Включите TCP/IP режим (устройство должно быть подключено по USB)
adb tcpip 5555

# 2. Подключитесь по IP (замените IP на адрес вашего устройства)
adb connect 192.168.1.100:5555

# 3. Проверьте подключение
adb devices
```

### Шаг 6: Проверка

После настройки проверьте:
```bash
adb devices
```

Должно показать:
```
List of devices attached
192.168.1.100:5555    device
```

Теперь можно отключить USB кабель!

### Использование

После настройки запустите:
```bash
/opt/grani/dev_run.sh
```

Скрипт автоматически найдет устройство по сети.

## Решение проблем

### Устройство не видно в `adb devices`

1. **Проверьте USB подключение:**
   - Убедитесь, что кабель подключен
   - Попробуйте другой USB порт
   - Попробуйте другой кабель

2. **Проверьте настройки устройства:**
   - Отладка по USB включена?
   - Разрешена отладка с этого компьютера?

3. **Перезапустите ADB:**
   ```bash
   adb kill-server
   adb start-server
   adb devices
   ```

### Не удается подключиться по сети

1. **Устройство и компьютер в одной Wi-Fi сети?**
   - Проверьте, что оба подключены к одной сети

2. **Правильный IP адрес?**
   - Убедитесь, что IP адрес актуальный
   - IP может измениться при переподключении к Wi-Fi

3. **Firewall блокирует порт?**
   - Проверьте, что порт 5555 не заблокирован

4. **Попробуйте переподключиться:**
   ```bash
   adb disconnect 192.168.1.100:5555
   adb connect 192.168.1.100:5555
   ```

### Устройство отключается

- Убедитесь, что Wi-Fi не отключается
- Проверьте настройки энергосбережения
- Для Android 11+ используйте встроенную "Беспроводную отладку"

## Полезные команды

```bash
# Список всех устройств
adb devices

# Отключить сетевое устройство
adb disconnect 192.168.1.100:5555

# Подключить сетевое устройство
adb connect 192.168.1.100:5555

# Перезапустить ADB сервер
adb kill-server && adb start-server

# Получить IP адрес устройства (если подключено по USB)
adb shell "getprop dhcp.wlan0.ipaddress"
```

## Для Android 11+ (беспроводная отладка)

Если у вас Android 11 или новее, можно использовать встроенную функцию:

1. **На устройстве:**
   - Настройки → Для разработчиков → Беспроводная отладка
   - Включите "Беспроводная отладка"
   - Нажмите "Параметры беспроводной отладки"
   - Запишите IP адрес и порт (например, `192.168.1.100:12345`)

2. **На сервере:**
   ```bash
   /opt/grani/setup_adb_network.sh --skip-usb --ip 192.168.1.100 --port 12345
   ```

Или напрямую:
```bash
adb connect 192.168.1.100:12345
```
