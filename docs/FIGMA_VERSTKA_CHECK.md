# Проверка верстки экранов по макетам Figma

**Дата:** 2025-01-XX  
**Статус:** ⏳ Требует настройки MCP Figma сервера

## Настройка MCP Figma сервера

### Требования
- MCP сервер Figma должен быть настроен в Cursor согласно `FIGMA_MCP_SETUP_COMPLETE.md`
- Токен Figma: `FIGMA_PERSONAL_ACCESS_TOKEN`
- Node.js должен быть установлен

### Конфигурация
Файл конфигурации Cursor (Linux):
```
~/.config/Cursor/User/globalStorage/saoudrizwan.claude-dev/settings/cline_mcp_settings.json
```

Добавить:
```json
{
  "mcpServers": {
    "figma": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-figma"
      ],
      "env": {
        "FIGMA_PERSONAL_ACCESS_TOKEN": "FIGMA_PERSONAL_ACCESS_TOKEN"
      }
    }
  }
}
```

### Проверка доступа
После настройки и перезапуска Cursor проверить доступ к страницам:
- Auth Flow: `515:201`
- VPN Flow: `562:732`
- Документация: `574:811`
