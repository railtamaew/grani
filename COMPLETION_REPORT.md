## 📝 Следующие шаги (для пользователя)

1. **Выполнить миграцию секретов:**
   ```bash
   sudo ./backend/scripts/safe_secrets_migration.sh
   ```
   См. `MIGRATION_GUIDE.md` для деталей

2. **Опционально: Ротировать SECRET_KEY:**
   ```bash
   python3 backend/scripts/rotate_secrets.py --generate-new-secret-key
   ```
   См. `docs/SECRETS_MANAGEMENT.md` для деталей

3. **Обновить конфигурацию приложения:**
   - Docker Compose: добавить `env_file: /etc/grani/secrets.env`
   - Systemd: добавить `EnvironmentFile=/etc/grani/secrets.env`
